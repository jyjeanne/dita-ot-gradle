Installation

How to add the DITA-OT Gradle Plugin to your project.

The plugin is published to the Gradle Plugin Portal and can be easily added to your project.

Prerequisites

  • Gradle 6.5 or later (Gradle 9.0+ recommended)
  • Java 8 or later
  • DITA-OT 3.0 or later

Groovy DSL

Add the following to your build.gradle:

plugins {
    id 'io.github.jyjeanne.dita-ot-gradle' version '2.3.2'
}

Kotlin DSL

Add the following to your build.gradle.kts:

plugins {
    id("io.github.jyjeanne.dita-ot-gradle") version "2.3.2"
}

Verify Installation

After adding the plugin, verify it's available by running:

./gradlew tasks --group="DITA-OT"

You should see the dita task listed.
