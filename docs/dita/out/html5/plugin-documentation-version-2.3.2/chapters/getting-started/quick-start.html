Quick Start

Get started with DITA-OT Gradle Plugin in minutes.

Step 1: Add the Plugin

Add the plugin to your build.gradle.kts (Kotlin DSL):

plugins {
    id("io.github.jyjeanne.dita-ot-gradle") version "2.4.0"
}

Or in build.gradle (Groovy DSL):

plugins {
    id 'io.github.jyjeanne.dita-ot-gradle' version '2.4.0'
}

Step 2: Download DITA-OT (Optional)

The plugin can automatically download DITA-OT for you:

// Kotlin DSL
val downloadDitaOt by tasks.registering(DitaOtDownloadTask::class) {
    version.set("4.2.3")
}

// Groovy DSL
tasks.register('downloadDitaOt', DitaOtDownloadTask) {
    version = '4.2.3'
}

This downloads DITA-OT to build/dita-ot/dita-ot-4.2.3.

Step 3: Configure the Task

Configure the dita task with your input files:

// Kotlin DSL
dita {
    ditaOt(layout.buildDirectory.dir("dita-ot/dita-ot-4.2.3"))
    input("docs/guide.ditamap")
    transtype("html5")
}

// Groovy DSL
dita {
    ditaOt layout.buildDirectory.dir('dita-ot/dita-ot-4.2.3')
    input 'docs/guide.ditamap'
    transtype 'html5'
}

Step 4: Run the Build

Execute the transformation:

./gradlew downloadDitaOt dita

The output will be generated in the build directory.

Complete Example (Kotlin DSL)

Here's a complete build.gradle.kts example:

plugins {
    id("io.github.jyjeanne.dita-ot-gradle") version "2.4.0"
}

// Download DITA-OT automatically
val downloadDitaOt by tasks.registering(DitaOtDownloadTask::class) {
    version.set("4.2.3")
}

// Configure transformation
dita {
    ditaOt(layout.buildDirectory.dir("dita-ot/dita-ot-4.2.3"))
    input("docs/guide.ditamap")
    output("build/docs")
    transtype("html5")

    properties {
        "nav-toc" to "partial"
        "args.copycss" to "yes"
    }
}

// Make dita depend on download
tasks.named("dita") {
    dependsOn(downloadDitaOt)
}

Installing DITA-OT Plugins

You can also install additional DITA-OT plugins:

val installPlugins by tasks.registering(DitaOtInstallPluginTask::class) {
    dependsOn(downloadDitaOt)
    ditaOtDir.set(layout.buildDirectory.dir("dita-ot/dita-ot-4.2.3"))
    plugins.set(listOf("org.dita.pdf2", "com.example.custom-plugin"))
}
