Upgrade Recommendations

To get the most from version 2.3.x:

  1. Enable Configuration Cache in gradle.properties
  2. Use ditaProperties.put() instead of closures
  3. Upgrade to Gradle 9.0 or later
  4. Enable parallel execution with org.gradle.parallel=true
