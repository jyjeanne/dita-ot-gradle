Developing Custom Plugins

Fast iteration workflow for DITA-OT plugin development.

The examples/custom-plugin-dev project provides a complete plugin development environment.

Quick Start

cd examples/custom-plugin-dev

# Run development build (download DITA-OT, install plugin, transform)
./gradlew dev

# Auto-rebuild when plugin source changes
./gradlew dev --continuous

Project Structure

custom-plugin-dev/
├── build.gradle
├── src/
│   └── my-custom-plugin/
│       ├── plugin.xml
│       ├── css/
│       └── xsl/
└── test-content/
    ├── test.ditamap
    └── topics/

Development Workflow

  1. Initial Setup: ./gradlew extractDitaOt
  2. Development: ./gradlew dev --continuous
  3. Debug: ./gradlew installMyPlugin --info
  4. Clean Rebuild: ./gradlew cleanAll dev

Tips

  • Use --continuous mode for rapid iteration
  • Check logs with --info when debugging
  • Test with different DITA-OT versions using -PditaOtVersion
