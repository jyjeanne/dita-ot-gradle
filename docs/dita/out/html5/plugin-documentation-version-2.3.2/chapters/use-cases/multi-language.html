Multi-Language Documentation

Build documentation in multiple languages with parallel execution.

The examples/multi-language project demonstrates how to build localized documentation efficiently.

Quick Start

cd examples/multi-language

# Build all languages (parallel)
./gradlew buildAll

# Build specific language
./gradlew buildEn
./gradlew buildFr
./gradlew buildDe

Parallel Execution

Enable parallel builds in gradle.properties:

org.gradle.parallel=true
org.gradle.configuration-cache=true

This allows all language builds to run simultaneously, significantly reducing total build time.

Dynamic Task Generation

Tasks are created dynamically for each language:

val languages = listOf("en", "fr", "de")

languages.forEach { lang ->
    tasks.register<DitaOtTask>("build${lang.capitalize()}") {
        input(file("content/$lang/guide.ditamap"))
        output(layout.buildDirectory.dir("output/$lang"))
        transtype("html5")
    }
}
