The plugin includes several example projects demonstrating various use cases.

Running Examples

Each example can be run independently:

# Navigate to example directory
cd examples/simple

# Run the default task
./gradlew dita

# Or run with specific options
./gradlew dita --info

Customizing Examples

Examples are designed to be modified for your needs:

  1. Copy the example directory to a new location
  2. Update build.gradle or build.gradle.kts
  3. Replace sample DITA content with your own
  4. Adjust output settings as needed
