Introductions

A modern Gradle plugin for publishing DITA documents with DITA Open Toolkit.

The DITA-OT Gradle Plugin provides seamless integration between Gradle build automation and DITA Open Toolkit transformations. This plugin enables you to incorporate DITA documentation publishing into your standard Gradle build workflow.

What is DITA-OT?

DITA Open Toolkit (DITA-OT) is an open-source publishing engine for content authored in the Darwin Information Typing Architecture (DITA). It transforms DITA content into various output formats including:

  • HTML5
  • PDF
  • Markdown
  • Eclipse Help
  • And many more through plugins

Why Use This Plugin?

This plugin offers several advantages for documentation teams:

  • Configuration Cache Support - Up to 77% faster incremental builds
  • Modern Architecture - Built with Provider API for Gradle 9.0 compatibility
  • Cross-Platform - Works on Windows, macOS, and Linux
  • Full DSL Support - Both Groovy and Kotlin DSL configurations
  • Active Maintenance - Regular updates and bug fixes
